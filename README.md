# SudokuSolver
An advanced sudoku solving application.

# How to use
## 1. Download images of the sudokus you want to solved
![image](https://user-images.githubusercontent.com/74363934/120889126-03bbc580-c5fc-11eb-95d6-7c02d1a81235.png)

## 2. Start up the Sudoku-Solver
![image](https://user-images.githubusercontent.com/74363934/120889229-61501200-c5fc-11eb-8b79-bcecb056cc55.png)

## 3. Navigate to the folder your sudokus are in
![Screenshot 2021-06-05 124958](https://user-images.githubusercontent.com/74363934/120889427-521d9400-c5fd-11eb-8449-14dfab41c645.png)

## 4. Press Open
![image](https://user-images.githubusercontent.com/74363934/120889443-6d889f00-c5fd-11eb-8e83-11b20f7e8cb8.png)

### It should now look like this
![image](https://user-images.githubusercontent.com/74363934/120889484-b6405800-c5fd-11eb-861e-e017cb335d1f.png)

## 5. Choose how many Threads should be used
![image](https://user-images.githubusercontent.com/74363934/120889452-8bee9a80-c5fd-11eb-8ccd-c234fb3f16fe.png)

## 6. Solve your sudoku with one click
![image](https://user-images.githubusercontent.com/74363934/120889514-d1ab6300-c5fd-11eb-9dd3-5cf133d5e865.png)

## 7. Enjoy!
